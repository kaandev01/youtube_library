# 📚 youLIB v2 — YouTube Kişisel Bilgi Kütüphanesi

YouTube videolarını AI özetiyle arşivle, klasörlere ayır, puanla, çoklu dil ve özet modu seçenekleriyle yönet.

## 🆕 v2 Yenilikleri

- **Klasör Sistemi** — Video organizasyonu, özel klasörler
- **Çoklu Özet Modları** — Kısa / Detaylı / Profesyonel Rapor / Zaman Bloklu
- **Önemli Anlar** — 5 kritik an, 3 sınavlık nokta, 4 aksiyon önerisi
- **Çoklu Dil** — TR / EN / DE / FR / ES çıktı dili
- **Tema Sistemi** — Amber (siyah-sarı) ve Mono (siyah-beyaz) tema

## 🚀 Kurulum

1. `chrome://extensions/` sayfasına git
2. **Geliştirici modu**nu etkinleştir
3. **Paketlenmemiş öğe yükle** → Bu klasörü seç

## 🔑 API Keyleri

- **Groq** (AI): [console.groq.com/keys](https://console.groq.com/keys)
- **Supadata** (Transkript): [supadata.ai/dashboard](https://supadata.ai/dashboard)

## 📖 Kullanım

1. YouTube'da video aç
2. Sağdaki **📼 Arşivle** butonuna tıkla
3. Özet türü ve dili seç
4. Klasöre ata
5. Oluştur → Kaydet
